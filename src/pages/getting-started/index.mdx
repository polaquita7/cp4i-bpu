## Objetivos

- Adquirir experiencia práctica básica en demostrar la construcción de una API.


- Adquirir experiencia práctica básica en mostrar la experiencia de los consumidores de APIs en el portal de desarrolladores de API Connect.


– Desarrollar skills para planificar, configurar y ejecutar una actividad con cliente utilizando el Cloud Pak for Integration.


– Desarrollar el conocimiento necesario para demostrar el valor de las soluciones de IBM.


- Adquirir nociones básicas de la plataforma de integración y poder realizar recomendaciones de arquitectura.


- Conocer aspectos básicos de licenciamiento.
